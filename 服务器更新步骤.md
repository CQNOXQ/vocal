# 在阿里云服务器更新代码

## 步骤

### 1. 进入项目目录

```bash
cd /opt/vocal
```

### 2. 拉取最新代码

```bash
git pull
```

### 3. 重新构建

```bash
docker compose down
docker compose up -d --build
```

### 4. 查看日志

```bash
docker compose logs -f
```

## 如果还有错误

```bash
# 清理缓存
docker compose down -v
docker system prune -a

# 重新构建
docker compose up -d --build
```
